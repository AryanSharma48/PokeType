# PokéType 

**PokéType** is a Pokémon themed typing speed test website built with **REACT**.  
Type the random sentence as fast and accurately as you can — and watch your **WPM**, **accuracy**, and **time** update live!


---

## Check it out live: 
  ### *https://poketypist.vercel.app*
---

## ✨ Features

🔥 Random sentence every test  
⏱ Auto-start timer when typing begins  
📊 Real-time **WPM**, **accuracy**, and **time**  
🎨 Highlights correct / incorrect letters  
🔁 One-click restart for a new round  

---

## 🛠 Tech Stack

🎨 **Tailwind** – Styling & layout  
⚙️ **React with JavaScript** – Typing logic, stats & timer  

---

## ⚙️ How It Works

1. A random sentence is picked from a predefined list.  
2. The timer starts when you type your first character.  
3. Each character is compared live to the target text.  
4. WPM and accuracy update in real-time.  
5. Click **Restart** to try again with a new sentence.
